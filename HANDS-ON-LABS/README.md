# Hands-On Labs & Implementation Exercises

Practical experience in deploying and managing Windows Server infrastructure is essential for the AZ-800 exam.

## 🧪 Microsoft Learn Sandbox

The Microsoft Learn Sandbox provides a free, temporary Azure environment where you can deploy Windows Server infrastructure.

### How to Access the Sandbox
1. Navigate to any Microsoft Learn module that includes a hands-on exercise.
2. Sign in with your Microsoft account.
3. Click the **Activate sandbox** button.
4. Review the permissions and accept.
5. You will have a set amount of time to use the sandbox environment.

## 📝 Recommended Implementation Exercises

### Lab 1: Deploy Active Directory Domain Services
* **Objective**: Deploy AD DS domain controllers and configure a domain.
* **Microsoft Learn Guide**: [Deploy AD DS](https://learn.microsoft.com/en-us/training/modules/deploy-manage-active-directory-domain-services/)
* **Key Skills**: Domain controller deployment, forest and domain configuration, FSMO roles.

### Lab 2: Configure DNS and DHCP
* **Objective**: Deploy and configure DNS and DHCP servers.
* **Microsoft Learn Guide**: [Configure DNS and DHCP](https://learn.microsoft.com/en-us/training/modules/manage-windows-server-networking/)
* **Key Skills**: DNS zone configuration, DHCP scope management, IP address allocation.

### Lab 3: Implement Storage Solutions
* **Objective**: Configure storage pools and create virtual disks.
* **Microsoft Learn Guide**: [Implement Storage Spaces](https://learn.microsoft.com/en-us/training/modules/implement-storage-spaces/)
* **Key Skills**: Storage pool creation, virtual disk configuration, storage tiering.

### Lab 4: Configure Failover Clustering
* **Objective**: Create a failover cluster and configure cluster resources.
* **Microsoft Learn Guide**: [Configure failover clustering](https://learn.microsoft.com/en-us/training/modules/configure-failover-clustering/)
* **Key Skills**: Cluster creation, resource configuration, failover testing.

### Lab 5: Implement Backup and Recovery
* **Objective**: Configure Windows Server Backup and perform recovery operations.
* **Microsoft Learn Guide**: [Implement backup and recovery](https://learn.microsoft.com/en-us/training/modules/implement-backup-recovery-windows-server/)
* **Key Skills**: Backup configuration, recovery procedures, backup scheduling.
