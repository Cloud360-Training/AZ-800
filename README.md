# AZ-800 Microsoft Windows Server Hybrid Core Infrastructure Administrator: Ninja Training

Welcome to the **AZ-800 Microsoft Windows Server Hybrid Core Infrastructure Administrator Ninja Training**! This repository is your comprehensive resource for preparing for the AZ-800 certification exam, Microsoft's foundational certification for Windows Server administrators.

Following the Microsoft Ninja Training format, this guide takes you from core infrastructure fundamentals to advanced hybrid deployment and management through a carefully structured learning path.

## 🎯 Training Philosophy

This training program prioritizes resources in the following order to ensure you get the most accurate and effective preparation:
1. **Microsoft Official Documentation & Learn Modules** (Highest Priority)
2. **Cloud360 Training Videos & Resources**
3. **Hands-on Labs & Sandbox Exercises**
4. **Community Resources & Practice Tests**

## 📚 Course Structure

The training is organized into four modules that directly map to the official AZ-800 exam skills measured.

### [Module 1: Deploy and Manage Active Directory Domain Services (30-35%)](./1-AD-DS-DEPLOYMENT/README.md)
* Deploy and manage AD DS domain controllers
* Configure and manage multi-site, multi-domain, and multi-forest environments
* Create and manage AD DS security principals
* Implement and manage hybrid identities

### [Module 2: Manage Windows Server Networking (20-25%)](./2-NETWORK-INFRASTRUCTURE/README.md)
* Configure and manage IP addressing
* Implement DNS
* Configure and manage DHCP
* Implement and manage network connectivity

### [Module 3: Implement Storage Solutions (15-20%)](./3-STORAGE-REPLICATION/README.md)
* Configure and manage storage
* Implement data deduplication
* Configure and manage replication
* Implement Storage Spaces

### [Module 4: Implement Disaster Recovery and High Availability (20-25%)](./4-DISASTER-RECOVERY/README.md)
* Implement backup and recovery
* Implement high availability
* Configure failover clustering
* Implement Hyper-V replication

## 🎓 Prerequisites

Before starting AZ-800, you should have:
* Strong understanding of Windows Server operating system
* Experience with Active Directory administration
* Knowledge of networking concepts
* Familiarity with Azure services
