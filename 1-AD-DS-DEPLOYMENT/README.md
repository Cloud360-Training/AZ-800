# Module 1: Deploy and Manage Active Directory Domain Services (30-35%)

This module covers deploying and managing Active Directory Domain Services in on-premises and hybrid environments.

## 📖 Microsoft Official Documentation & Learn

### 1. Deploy and Manage AD DS Domain Controllers
* **Microsoft Learn Path**: [Deploy and manage AD DS domain controllers](https://learn.microsoft.com/en-us/training/modules/deploy-manage-active-directory-domain-services/)
* **Key Topics**:
  * Deploy domain controllers on-premises
  * Deploy domain controllers in Azure
  * Deploy read-only domain controllers (RODCs)
  * Manage FSMO roles
  * Troubleshoot domain controller issues

### 2. Configure and Manage Multi-Site, Multi-Domain, and Multi-Forest Environments
* **Microsoft Learn Path**: [Configure multi-site AD DS](https://learn.microsoft.com/en-us/training/modules/configure-manage-multi-site-environments/)
* **Key Topics**:
  * Configure forest and domain trusts
  * Configure AD DS sites
  * Configure AD DS replication
  * Manage replication topology
  * Troubleshoot replication issues

### 3. Create and Manage AD DS Security Principals
* **Microsoft Learn Path**: [Create and manage AD DS users and groups](https://learn.microsoft.com/en-us/training/modules/create-manage-ad-ds-security-principals/)
* **Key Topics**:
  * Create and manage users and groups
  * Manage service accounts
  * Join servers to AD DS
  * Manage group policies

### 4. Implement and Manage Hybrid Identities
* **Microsoft Learn Path**: [Implement hybrid identities](https://learn.microsoft.com/en-us/training/modules/implement-manage-hybrid-identities/)
* **Key Topics**:
  * Integrate Microsoft Entra ID with AD DS
  * Implement Microsoft Entra Connect Sync
  * Implement Microsoft Entra Cloud Sync
  * Manage authentication in hybrid environments
