# Module 2: Manage Windows Server Networking (20-25%)

This module covers configuring and managing Windows Server networking infrastructure.

## 📖 Microsoft Official Documentation & Learn

### 1. Configure and Manage IP Addressing
* **Microsoft Learn Path**: [Configure IP addressing](https://learn.microsoft.com/en-us/training/modules/manage-windows-server-networking/)
* **Key Topics**:
  * Configure IPv4 and IPv6 addressing
  * Configure IPAM
  * Manage IP address allocation

### 2. Implement DNS
* **Microsoft Learn Path**: [Implement DNS](https://learn.microsoft.com/en-us/training/modules/implement-dns-windows-server/)
* **Key Topics**:
  * Deploy DNS servers
  * Configure DNS zones
  * Manage DNS records
  * Configure DNS security

### 3. Configure and Manage DHCP
* **Microsoft Learn Path**: [Configure DHCP](https://learn.microsoft.com/en-us/training/modules/configure-manage-dhcp/)
* **Key Topics**:
  * Deploy DHCP servers
  * Configure DHCP scopes
  * Manage DHCP options
  * Implement DHCP failover

### 4. Implement and Manage Network Connectivity
* **Microsoft Learn Path**: [Implement network connectivity](https://learn.microsoft.com/en-us/training/modules/implement-network-connectivity/)
* **Key Topics**:
  * Configure network adapters
  * Implement VPN
  * Configure remote access
